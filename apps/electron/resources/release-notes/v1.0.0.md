# v1.0.0 — First Release

---

## Welcome to Mengko Agents

Mengko Agents is an open-source desktop application that provides an agent-native interface for working with Claude and other LLMs.

### New Identity

- **Rebranded from Craft Agents** — The project has been renamed to Mengko Agents with a fresh new identity
- **New data directory** — Configuration and data now stored in `~/.mengko-agent/`
- **New App ID** — `com.mengko.agents`

### Features

- **Multi-provider support** — Connect to Anthropic Claude, OpenAI, Google, OpenRouter, Ollama, and any OpenAI-compatible endpoint
- **ChatGPT Plus & GitHub Copilot** — Sign in with your existing subscriptions
- **Session management** — Organize your conversations with labels, statuses, and search
- **MCP integration** — Connect to Linear, GitHub, Slack, and more through MCP servers
- **Multiple modes** — Explore mode for safe reading, Ask to Edit for confirmations, Auto mode for full autonomy
- **Internationalization** — English and Chinese language support
- **Dark/Light themes** — Customizable appearance with multiple color themes
- **Skills** — Define custom agent instructions for specialized workflows
- **Hooks** — Event-driven automation for advanced workflows

### Data Directory

All user data is stored in `~/.mengko-agent/`:

```
~/.mengko-agent/
├── config.json        — App configuration
├── preferences.json   — User preferences
├── credentials.enc   — Encrypted API keys
├── workspaces/       — Workspace folders
│   └── my-workspace/
│       ├── sessions/  — Chat sessions
│       ├── sources/   — MCP sources
│       └── skills/    — Custom skills
├── themes/           — Color themes
├── docs/             — Documentation
└── release-notes/   — Update notes
```

### Migration from Craft Agents

If you have existing data in `~/.craft-agent/`, you can manually copy it:

```bash
cp -r ~/.craft-agent ~/.mengko-agent
```

Then update `~/.mengko-agent/config.json` to change workspace paths from `.craft-agent` to `.mengko-agent`.
